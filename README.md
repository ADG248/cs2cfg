# Counter-Strike 2 Configuration File - cs2cfg

**Counter-Strike 2 Configuration File (autoexec)**<br/>
This is a base autoexec file for an easier setup from scratch.<br/>

### Place autoexec.cfg in \Steam\steamapps\common\Counter-Strike Global Offensive\game\csgo\cfg
```
\Steam\steamapps\common\Counter-Strike Global Offensive\game\csgo\cfg
```
### Enter "+exec autoexec" in CS2's Launch Options
```
+exec autoexec
```

### Additional Notes
- Type "exec autoexec" in the console to update your changes in-game. You can also bind a key "bind *key* exec autoexec".<br/>
- Settings changed in-game will reset back to the Config File's value the each time it is loaded.<br/>
- This Config also includes things such as PracticeMode On/Off Keybinds, Delete Grenade Keybind (Delete Smokes, etc.)<br/>
